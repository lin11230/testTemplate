Glossary

Traffic Savings Bandwidth savings by having Nexusguard manage your traffic.

Bandwidth Usage Netflow uses 95th percentile to meter monthly bandwidth usage (calendar days).

Cache Ratio Cached hits divided by legitimated reqeusts.
(Note: Only HTTP status codes 200 and 304 are counted.)

Cache Hits The accumulated number of HTTP requests served from cache.

Legitimate Requests The accumulated number of legitimate requests, including authorized web crawlers, of the day.

Served by Origin Traffic that has flowed out of the backend server.

Threats = Blocked + Suscipious Requests

Mitigated by Nexusguard Requests blocked and logged by Nexusguard.

Request Requests recorded by access log.

Visitor The number of unique visitors counted every minute, hour and day. In the absence of cookies, unique visitors are tracked using the hash generated by the IP address, geolocation, browser version, OS platform, etc.

Page Views All static pages (html, htm) under HTTP status code 200 and dynamic pages (php, asp, do, jsp, py, and so forth).

Visitor Breakdown by No. of Visits Distribution of numbers of visits made by unique visitors, i.e. how many visitors have made one visit, how many visitors have made two visits, and how many visitors have made N visitors within a given period of time.

HTTP Status Distribution of HTTP status codes recorded by access log.

Platform OS platforms used by visitors.
